staticland

A home for your static sites.

Features

  • One command to deploy a site.
  • Automatic SSL using Let's Encrypt.
  • Use any static site generator.
  • Host it yourself. Use it how you want.

    • staticland on github

Usage

Login once:

    staticland login \
      --email wat@static.land \
      --password thisisnotapassword \
      --server https://api.static.land

One command to deploy a site with SSL:

    staticland deploy \
      --source /path/to/site \
      --domain example.com

Get started

The static.land hosted service is currently in a closed beta.

This gives you two options:

  • Hire us to set up a staticland server for you, including any unique needs you have for your use case.
  • Use the open source code to host a staticland server yourself. github.com/staticland
