staticland
Github

Static sites with free, automatic SSL for custom domains.

npm install --global staticland
staticland register
staticland site/ yourdomain.com

Features

  • One command to deploy a site.
  • Automatic SSL using Let's Encrypt.
  • Use a 200.html file for HTML5 pushstate support.
  • Use any static site generator.
  • Open source. Host it yourself to use it how you want.

Usage

Create an account:

staticland register

Use --server yourstaticlandserver.com to use a custom server.

Set up DNS:

Ensure that your domain's DNS settings are pointing at 52.39.104.182 (or the IP address of your custom server) before deploying: 

host static.land
static.land has address 52.39.104.182

Setting the DNS before deploying is required for Let's Encrypt to successfully provision a certificate.

Deploy a site with auto-SSL:

staticland site/ example.com

Advanced usage

The static.land hosted service is limited. You can publish static sites with automatic SSL provisioned via Let's Encrypt. That's it.

Need more features, a private, managed server, or advanced integration with existing systems? You have two options:

  • Hire us to set up an internal static site deployment system that fits your use case.
  • Use the open source code to host a staticland server yourself. github.com/staticland
